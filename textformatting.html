• Text-color

    • Text-color property is used to set the color of the text.

  • Text alignment

    • Text alignment property is used to set the horizontal alignment of the text. The text can be set to left, right, centered and justified alignment.

  • Text decoration

    • Text decoration is used to add or remove decorations from the text. Text decoration can be underline, overline, line-through or none.

  • Text transformation

    • Text transformation property is used to change the case of text, uppercase or lowercase. Text transformation can be uppercase, lowercase or capitalise .

  • Text indentation

    • Text indentation property is used to indent the first line of the paragraph. The size can be in px, cm, pt.

  • LETTER SPACING

    • This property is used to specify the space between the characters of the text. The size can be given in px.

  • LINE HEIGHT

    • This property is used to set the space between the lines.

  • TEXT DIRECTION

    • Text direction property is used to set the direction of the text. The direction can be set by using rtl : right to left .

  • WORD SPACING

    • Word spacing is used to specify the space between the words of the line. The size can be given in px.
